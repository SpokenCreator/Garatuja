## Revisão de WebDesign

### [HTML]
HTML (HyperText Markup Language) é a linguagem de marcação que define a **estrutura e o conteúdo** de uma página web. Ele não é uma linguagem de programação, não tem lógica, condições ou loops. É simplesmente um conjunto de tags que dizem ao navegador o que cada pedaço de conteúdo representa: isso é um título, isso é um parágrafo, isso é uma imagem, isso é um link.

---

### [CSS]
CSS (Cascading Style Sheets) é a linguagem responsável pela **aparência visual** de tudo que o HTML estruturou. Com CSS você define cores, fontes, espaçamentos, tamanhos, layouts e animações. Se o HTML é o esqueleto, o CSS é a roupa e o design.

---

### [JS]
JavaScript é a única linguagem de programação nativa dos navegadores, e é ela que traz **interatividade e comportamento** à web. Enquanto HTML e CSS são estáticos, descrevem como as coisas são, o JavaScript permite que as coisas mudem em resposta a ações do usuário ou a dados externos. Com JS você pode modificar o HTML e o CSS em tempo real, reagir a cliques e digitação, criar animações complexas e muito mais.
